Import-KSExcelWorksheet
 
Synopsis

Import from Excel using the Microsoft Access Database Engine.

Syntax

Import-KSExcelWorksheet [-FileName] <String> [-WorksheetName <String>] [-FirstDataRow <UInt32>] [-HeaderRowNumber <UInt32>] [-IgnoreEmptyRows] [<CommonParameters>]

Import-KSExcelWorksheet [-FileName] <String> [-WorksheetName <String>] [-FirstDataRow <UInt32>] -Header <String[]> [-IgnoreEmptyRows] [<CommonParameters>]

Import-KSExcelWorksheet [-FileName] <String> [-WorksheetName <String>] [-FirstDataRow <UInt32>] -TypedHeader <OrderedDictionary> [-IgnoreEmptyRows] [<CommonParameters>]

Description

Import-KSExcelWorksheet uses the Microsoft Access Database Engine to read from Excel files. The Microsoft Access Database Engine can read from Excel without needing to use the Excel COM Object.

Select-KSString
 
Synopsis

Select a matching string from an alphabetically sorted file.

Syntax

Select-KSString [-String] <String> [-FileName] <String> [<CommonParameters>]

Description

Select-KSString is a specialised binary (half interval) searcher designed to find matches in sorted ASCII encoded text files.
