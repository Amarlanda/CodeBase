Get-KSCertificate
 
Synopsis

Get certificates from a local or remote certificate store.

Syntax

Get-KSCertificate [[-StoreName] <StoreName[]>] [[-StoreLocation] {CurrentUser | LocalMachine}] [[-ComputerName] <String>] [-HasPrivateKey] [-Expired] [<CommonParameters>]

Description

Get X509 certificates from a certificate store.

Install-KSCertificate
 
Synopsis

Install a certificate.

Syntax

Install-KSCertificate [-Certificate] <X509Certificate2> [[-StoreName] {AddressBook | AuthRoot | CertificateAuthority | Disallowed | My | Root | TrustedPeople | TrustedPublisher}] [[-StoreLocation] {CurrentUser | LocalMachine}] [[-ComputerName] <String>] [<CommonParameters>]

Description

Test-KSTrustedCertificate
 
Synopsis

Test for a certificate in the TrustedPeople store on the target computer.

Syntax

Test-KSTrustedCertificate [-Certificate] <X509Certificate2> [[-StoreLocation] {CurrentUser | LocalMachine}] [[-ComputerName] <String>] [-Detail] [<CommonParameters>]

Description

Test-KSTrustedCertificate attempts to find a matching certificate in the TrustedPeople store.
